# 🏀 Basketball Scoreboard

A simple basketball scoreboard built with HTML, CSS, and JavaScript.

## Features

- Track scores for two teams (Neon Wolves vs Shadow Hawks)
- Add +1, +2, +3 points per team
- Reset scores back to zero
- Responsive design with Orbitron font

## Built With

- HTML
- CSS (Flexbox, clamp() for responsive sizing)
- JavaScript (DOM manipulation)

## What I Learned

- DOM manipulation (getElementById, innerText)
- Event handling (onclick)
- CSS Flexbox layout
- Responsive typography with clamp()
- Google Fonts integration
